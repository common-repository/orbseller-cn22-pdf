This plugin works with WooCommerce and will print a CN22 form in PDF format.

== Description ==

When sending commercial mail from the UK which has a value over Â£270 then you must include a customs form CN22. This plugin integrates with WooCommerce by adding an action to the orders page to print the CN22 form.

For the CN22 form to collate the correct information you will need to ensure that the shipping weight is set for each product else the weight will be 0.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Go to an Order and under Actions you will see a new action called Print CN22 Form.
